Live Asset Tracking

Track any number of assets, anywhere in the world.

Quartic can be used to track any asset that produces geospatial data. If the data are produced at approximate or long intervals, the platform will produce a ‘best estimate’ between updates. This allows organisations from security to freight carriers to know where everything they care about is, all the time.

The positional data can be used to drive further logic, for example, live spacing of transport vehicles. The data appears in all our front ends and can be shared with customers.

Geofencing

Any feature in the data can act as a geofence, and you can geofence multiple features at the same time.

Quartic lets you select any feature or collection of features that are relevant to you and trigger an event (an email alert or something much more complex, like a pricing change) based on live activity around the features of interest.

This can be fully automated and included in your business logic as a simple set of rules, all the way through to a complex probabilistic model.
